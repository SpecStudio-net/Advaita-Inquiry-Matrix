
# AIM Student State Model

This document defines the **student states** used by the Advaita Inquiry Matrix (AIM).
A student state represents the current cognitive and existential condition of a seeker.
Pedagogical moves are selected based on the detected state.

The model is inspired by traditional Advaita pedagogy, especially the guru–śiṣya
dialogues in the Upaniṣads and the methods used by Śaṅkarācārya.

---

# State 0 — Existential Confusion

**Description**  
The student experiences suffering, confusion, or dissatisfaction with life,
but does not yet clearly understand the nature of the problem.

Typical statements:

- “Something feels wrong in my life.”
- “I feel incomplete.”
- “I want peace.”

Recommended pedagogical focus:

- existential questioning
- introduction to Self-knowledge
- teacher–student relationship

Suggested clusters:

```
existential_inquiry
guru_transmission
motivation_for_inquiry
```

---

# State 1 — Philosophical Curiosity

**Description**  
The student begins to inquire into the nature of reality or consciousness.

Typical statements:

- “What is the Self?”
- “Is consciousness fundamental?”

Recommended pedagogical focus:

- conceptual introduction to Advaita
- cosmological and metaphysical discussion

Suggested clusters:

```
brahman_definition
cosmic_contemplation
identity_analysis
```

---

# State 2 — Misidentification

**Description**  
The student still identifies with body, mind, or personal story.

Typical statements:

- “I am this body.”
- “My thoughts define me.”

Recommended pedagogical moves:

- adhyāsa correction
- neti-neti analysis
- kośa discrimination

Suggested clusters:

```
adhyasa_correction
identity_analysis
negation_method
```

---

# State 3 — Witness Recognition

**Description**  
The student begins to recognize awareness as distinct from mental activity.

Typical statements:

- “I can observe my thoughts.”
- “There is a witnessing presence.”

Recommended pedagogical moves:

- sākṣī recognition
- contemplative inquiry

Suggested clusters:

```
witness_recognition
direct_inquiry
contemplative_inquiry
```

---

# State 4 — Identity Inquiry

**Description**  
The student investigates the identity of the witness and Brahman.

Typical statements:

- “Is the witness the ultimate Self?”
- “What is the relation between Self and Brahman?”

Recommended pedagogical moves:

- mahāvākya teaching
- identity revelation

Suggested clusters:

```
mahavakya_revelation
nondual_identity
```

---

# State 5 — Recognition of Nonduality

**Description**  
The student understands the teaching intellectually and may glimpse its truth.

Typical statements:

- “I see that awareness is not separate.”
- “The Self appears universal.”

Recommended pedagogical moves:

- contemplative stabilization
- removal of subtle doubts

Suggested clusters:

```
realization_integration
subtle_doubt_resolution
```

---

# State 6 — Stabilized Knowledge

**Description**  
The student abides in nondual understanding and expresses it naturally in life.

Typical expressions:

- spontaneous peace
- absence of existential fear
- natural compassion

Recommended pedagogical focus:

- integration into life
- ethical expression of realization

Suggested clusters:

```
ethical_expression_of_realization
liberated_life
cosmic_identity
```

---

# State Transition Model

Typical progression:

```
existential_confusion
↓
philosophical_curiosity
↓
misidentification_detected
↓
witness_recognition
↓
identity_inquiry
↓
nondual_recognition
↓
stabilized_knowledge
```

Transitions are **not strictly linear**. Students may move back and forth
between states as doubts arise and are resolved.

---

# AIM Teaching Logic

The AIM system operates by:

1. Detecting the student’s current state.
2. Selecting an appropriate pedagogical move.
3. Retrieving supporting passages from the corpus.
4. Guiding the inquiry forward.

This structure models the classical Advaita teaching dialogue found in
the Upaniṣads.
